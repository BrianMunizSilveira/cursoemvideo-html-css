Exercício 1: Estilização Básica de Caixas

Instruções:

  1. Crie um arquivo HTML básico.
  2. Adicione três elementos <div> dentro do <body>.
  3. Estilize cada <div> para ter largura de 200 pixels, altura de 100 pixels e uma borda sólida de 2 pixels. Use cores diferentes para cada div.
  4. Certifique-se de que haja algum espaço entre as caixas.

Exercício 2: Centralizar Conteúdo em uma Caixa

Instruções

  1. Crie um arquivo HTML básico.
  2. Adicione um elemento <div> dentro do <body>.
  3. Estilize esse <div> para ter uma largura de 400 pixels, uma altura de 200 pixels, uma borda sólida de 2 pixels e um fundo cinza claro.
  4. Centralize o texto dentro dessa caixa vertical e horizontalmente.
