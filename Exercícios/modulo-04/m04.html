  • Ex24

    Iframe (Quadros em Linha)

    Um iframe (inline frame) é uma "janela aberta" dentro da página que permite apresentar conteúdo de outras páginas. É um recurso presente desde as versões anteriores do HTML e ainda utilizado em diversos sites modernos.

  • Ex25

    Formulários

    Os formulários são fundamentais para interação em sites, permitindo o envio de informações como login, cadastros, buscas e muito mais. Este exercício explora a criação de formulários semânticos e acessíveis utilizando as principais tags e atributos do HTML5.

    Ver Exercício
  • Ex26

    Media Queries

    As Media Queries permitem criar estilos CSS adaptáveis para diferentes dispositivos e resoluções de tela. Neste capítulo, explore como personalizar a aparência de páginas web para smartphones, tablets, desktops e até impressoras, aplicando conceitos como design responsivo e a filosofia "Mobile First".
